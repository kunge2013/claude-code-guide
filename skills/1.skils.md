**新增功能**: 创建 Claude Code 自定义技能用于自动化 Git 提交流程

**核心功能**:
1. **智能分析变更**: 自动检查 `git status` 和 `git diff`，了解变更内容
2. **生成提交信息**: 基于变更类型和项目风格自动生成中文提交信息
3. **自动化提交**: 执行 `git add`、`git commit` 并添加 Co-Authored-By 标记
4. **可选推送**: 询问用户是否推送到远程仓库
5. **生成文档**: 自动记录每次提交的详细信息到本文件

**技术实现**:
- 使用 Claude Code Skills 机制创建可复用技能
- 配置文件: `skill.json` - 定义技能元信息
- 指令文件: `skill.md` - 定义详细的执行步骤和规则
- 遵循约定式提交规范 (Conventional Commits)

**支持的提交类型**:
- `add`: 新增功能或文件
- `fix`: 修复错误
- `update`: 更新功能
- `remove`: 删除功能或文件
- `refactor`: 代码重构
- `docs`: 文档变更
- `test`: 测试相关
- `style`: 代码风格调整

**安全特性**:
- 不修改 git config
- 不使用强制推送
- 不跳过 pre-commit hooks
- 推送前需用户确认
