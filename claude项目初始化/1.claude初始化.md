# Claude 项目初始化

## claude init 命令输出

运行 `claude init` 后，系统检测到当前是一个 Claude Code 指南项目，目录包含：

- **skills/** - Skills（技能）文档
- **simple-project/** - 一个简单项目示例
- **quick_key/** - 快捷键相关内容
- **plan/** - 规划材料
- **.claude/** - Claude 配置目录
- **claude项目初始化/** - Claude 项目初始化（新的未跟踪文件夹）
- 一些用于文档的 markdown 文件

系统询问想要做什么工作，可以帮助：

1. 探索或修改现有文档
2. 创建新内容
3. 使用 git 仓库
4. 其他任务

## 使用说明

`claude init` 命令用于初始化一个新的 Claude 项目或在现有项目中设置 Claude Code 环境。当在一个已有内容的目录中运行时，它会分析现有结构并提供针对性的建议。
